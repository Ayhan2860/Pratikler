## Insertion Sort 1. Dizi

| [22,27,16,2,18,6] -> Insertion Sort türüne göre aşamaları |
| ------------------------------------------------------------------|
| [22,27,16,2,18,6] |
| [22,27,16,2,18,6] |
| [16,22,27,2,18,6] |
| [2,16,22,27,18,6] |
| [2,16,18,22,27,6] |
| [2,6,16,18,22,27] |

+ Big-O Notation: O(n^2)  ----> O(36)   [n=6]
